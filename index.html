Комбинированные типы данных

структуры, которые позволяют объединять несколько простых типов данных в один составной объект

Используется для

  • Упрощения работы с множеством связанных данных
  • Сохранения целостности данных
  • Удобной сериализации данных для хранения в файлах или передачи по сети

Запись (структура данных)

объединение нескольких полей, каждое из которых может иметь разный тип данных

В Python представляются как словарь или класс

student = {
    "name": "Иван Иванов",
    "age": 18,
    "group": "ИСР22",
    "gpa": 4.7
}

Преимущества использования записей

  • Преимущества использования записей
  • Гибкость
  • Легкость сериализации

Пример записи сериализованных данных

				
import json

student = {
    "name": "Иван Иванов",
    "age": 18,
    "group": "ИСР22",
    "gpa": 4.7
}

with open("student.json", "w") as file:
    json.dump(student, file)

файлы и их виды

структурированное хранилище данных на диске или другом внешнем устройстве

примеры текстовых файлов

.txt - обычные текстовые файлы
.csv - файлы с разделителями, такие как запятые или табуляции
.json, .xml - файлы для хранения структурированных данных 

with open("example.txt", "w") as file:
		file.write("Hello, World!")

Файлы и их виды

Бинарные файлы

.bin - файлы с произвольными бинарными данными
.exe - исполняемые файлы
.jpg, .png - файлы изображений 

with open("example.bin", "wb") as file:
    file.write(b'\x00\x01\x02\x03')

Последовательный доступ

чтение или запись данных по порядку, начиная с начала файла и двигаясь до конца


Прямой доступ

обращаться к любому месту в файле без необходимости проходить предыдущие данные

Чтение данных из файла построчно


# Открытие файла в режиме чтения
with open("example.txt", "r") as file:

где r - чтение файла

Запись данных в файл


# Открытие файла в режиме записи (перезапись содержимого)
with open("example.txt", "w") as file:
    file.write("Первая строка\n")

где a - режим добавления, а w - режим записи

Синтаксис работы с файлами в Python

open()

открывает файл для чтения или записи

read()

считывает данные из файла

write()

записывает данные в файл

close()

закрывает файл (вручную, если не используется with)


# Чтение содержимого файла
with open("example.txt", "r") as file:
    content = file.read()
    print(content)
